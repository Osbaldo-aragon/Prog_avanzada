# 🤖 Machine Learning con Python - Ejercicios y Programas

Bienvenido a este repositorio dedicado a la práctica y aprendizaje de **Machine Learning (ML)** mediante **ejercicios, ejemplos y programas en Python**.

Aquí encontrarás material orientado a fortalecer fundamentos de programación, análisis de datos y modelos de aprendizaje automático, con un enfoque práctico para estudiantes, docentes y entusiastas de la ingeniería.

---

## 🎯 Objetivo del repositorio

Este repositorio tiene como propósito:

- Reunir ejercicios de **Machine Learning** desde nivel introductorio hasta intermedio.
- Compartir programas en **Python** aplicados a problemas reales y académicos.
- Servir como apoyo para clases, prácticas y proyectos de ingeniería.
- Promover el aprendizaje paso a paso con ejemplos claros y reproducibles.

---

## 🧠 Temas que se incluyen (y se irán agregando)

- Fundamentos de Python para ML
- Manejo de datos con `pandas` y `numpy`
- Visualización con `matplotlib`
- Preprocesamiento de datos
- Regresión lineal
- Regresión logística
- Árboles de decisión
- KNN (K-Nearest Neighbors)
- Métricas de evaluación
- División de datos (train/test)
- Ejercicios prácticos y mini proyectos

---

## 🛠️ Tecnologías y librerías utilizadas

- Python 3.x
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- Jupyter Notebook (opcional)
